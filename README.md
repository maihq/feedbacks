
# rubu.me 建议反馈区

哦亲爱的 [rubu.me](https://rubu.me/) 用户们！

欢迎通过这个渠道告知我们你的高见，比如说：

- 你最想看到什么新功能？
- 我们能为现有功能做什么优化或修正？
- 告诉我们你特喜欢哪个功能。
- 任何相关的建议或担忧。
- 等等事项……

进入 [issues](https://github.com/maihq/feedbacks/issues)，发送你的好点子吧！

你也可以选择在 twitter 上骚扰咱们的主要开发者 [@bitinn](https://twitter.com/bitinn)。

在此提前感谢你的参与。
